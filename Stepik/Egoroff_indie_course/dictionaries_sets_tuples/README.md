### Тип NoneType и значение None

Начнём новый раздел, посвящённый оставшимся неизученными типам данных. Начнём с одного из них — типа NoneType.

Во многих языках программирования, включая SQL, существует специальное значение, которое обозначает нейтральное или пустое состояние. В Python таким значением является None.
